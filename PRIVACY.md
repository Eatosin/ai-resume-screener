# Privacy Policy for TalentAlign

## Data Handling
*   **Ephemeral Processing:** Resumes are processed in-memory and discarded immediately after analysis.
*   **Persistence:** Only anonymized metadata (Score, Skills, Summary) is stored in the Supabase database for analytics.
*   **Third-Party:** Text extraction is performed via Google Gemini API. Please refer to Google's Data Privacy terms.

## Usage
This tool is for demonstration and portfolio purposes. Do not upload sensitive real-world documents without redaction.
